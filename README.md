# editorconfig-docset

Generating Dash docset version of http://editorconfig.org/ .

## Requirements

Ruby 2.2.2 or later.

## Usage

    git clone https://github.com/labocho/editorconfig-docset.git
    cd editorconfig-docset
    gem install bundler
    bundle install
    bundle exec rake install

### Updating

`bundle exec rake install` updates docset.
